# 豪嘉株式会社 二手奢侈品管理系统 V1

## 功能
- 商品录入
- 库存管理
- 古物台账
- EMS商业报关表
- 利润分析
- CSV导出

## 本地运行
```bash
npm install
npm run dev
```

## 打包
```bash
npm run build
```

## 部署到 Vercel
1. 注册 GitHub
2. 新建仓库
3. 上传本项目全部文件
4. 登录 Vercel
5. Import GitHub Repository
6. Deploy
7. 得到网址，例如 https://gouka-luxury-erp.vercel.app
